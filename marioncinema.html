Front-end Development Case Study

marion cinema

Combining the online movie theater web experience with tip-of-the-spear technology like Vue.js.

The objective of this project was to create a web app for a local dine-in cinema. The dashboard design and singular purpose made it a perfect candidate for Vue.js. The process of this project included user research, UI design and finally front-end development.
